FEATURE

最新の制作実績です。

作品タイトル：My Schedule
製作期間：1ヶ月

URL： https://my-schedule-kulala.herokuapp.com/

コメント：
スケジュールを管理できるサイトです。
自分のスケジュールを日にちごとに登録、管理することができます。
他ユーザーのスケジュールも共有することが可能です。
（お気に入り登録をして他ユーザーのスケジュールを確認しやすくすることもできます。）
【こだわったところ】
・ログイン認証を実装し、他ユーザーとスケジュールの共有ができるようにしました。
・ログイン後のトップページをカレンダー表示にし、その日の予定を閲覧、登録、更新、削除ができるようにしました。
・カレンダー表示の際も、スクロールをして登録したスケジュールが確認できるように設定をしました。
・前月、今月、次月ボタンと、年月を指定できるセレクトボックスを作成し、自分の確認したい年月にすぐ移動できるよう工夫しました。
・「お気に入り登録」機能を実装し、お気に入り登録をしたユーザーのスケジュールをすぐに確認できるよう「お気に入りユーザー一覧」ページを設置しました。
・他ユーザーのスケジュールは閲覧のみ行えるように設定し、自分のスケジュールのみを登録、更新、削除ができるようにルーティングを行いました。
Git repository : https://github.com/nekulala/my-schedule
